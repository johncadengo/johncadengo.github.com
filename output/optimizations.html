Pragmas

Some default values to adjust include:

  • synchronous
  • cache_size
  • temp_store

Also, when dealing with large data sets, and in the command line, set the following,

.mode csv
.output output.csv

This way it isn't output to the screen, but to a file. Note, however, that it will also redirect pragma outputs as well. So if you aren't seeing responses to your commands, they are in the output file.
